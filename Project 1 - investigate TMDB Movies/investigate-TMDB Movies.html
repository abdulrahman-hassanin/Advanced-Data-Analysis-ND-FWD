Introduction¶

  • How runtime correlate with the revenue , voting, popularity?
  • Does more budget cause more revenue?
  • Which genres are most popular?
  • Which genres has highest revenue?
  • What is the number of movies produced by companies?

Data Cleaning (Replace missing values - Remove duplicated rows - Fix datatypes)¶

After the overview lookup at the dataset:

  • We have 10866 examples of rows with 21 features of columns.
  • There is only one duplicated row.
  • There are 8874 rows has missing values
  • Columns sepreated by | are [keywords, cast, genres, production_companies].
  • Columns with missing values are ['imdb_id', 'cast', 'homepage', 'director', 'tagline', 'keywords', 'overview', 'genres', 'production_companies'].
  • All users are unique except one.
  • More than 50% of records have revenue and budget value as 0.

The Setps to cleaning the data will be as the following:

  1. Drop inconsistent columns
  2. Drop duplicated row
  3. Drop null rows
  4. Drop rows which have revenue and budget value as 0
  5. Fix datatype of release_data column

def get_relation(df,col_1, col_2):
    col_1_classes = df[col_1].unique()
    relation_dict = {}
    for c in col_1_classes:
        df1 = df[df[col_1] == c]
        relation_dict[c] = df1[col_2].sum()
        
    rel = pd.DataFrame(relation_dict.items(), columns=[col_1, col_2])
    rel.sort_values(by=[col_2], inplace=True, ascending=False)
    return rel
